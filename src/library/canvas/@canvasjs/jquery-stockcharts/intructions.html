CanvasJS jQuery StockCharts

CanvasJS jQuery StockChart Plugin for creating interactive charts & graphs for your web pages. Library supports a wide range of chart types including line, area, column, candlestick, etc.

Installing CanvasJS jQuery StockCharts

Download CanvasJS jQuery Charts

Download the commercial version of jQuery StockChart Package from My Account Page. Copy '@canvasjs' folder into src (lib or assets) folder of your app.

Import jQuery stockchart Plugin

Import jQuery StockCharts and jQuery into your application. The plugin has a dependency on jQuery, and hence that has to be imported as well. 

 import $ from "jquery";
 import './@canvasjs/stockchart/jquery-stockcharts';
Define a div (stockchart-container) for stockchart 
 <div id="stockChartContainer"></div>
Set the stockchart-options & create stockchart
